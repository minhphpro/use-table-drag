## Cài đặt

1. Clone repository:

   ```sh
   git clone https://github.com/minhphpro/use-table-drag
   cd use-table-drag
   ```

2. Cài đặt các dependencies:

   ```sh
   npm install
   ```

3. Chạy dự án:
   ```sh
   npm run dev
   ```
